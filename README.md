# Challenge-Alura-Store
 Challenge Alura Store - Proyecto de análisis de datos desarrollado con Python y Pandas, enfocado en comparar el desempeño de diferentes tiendas para determinar cuál ofrece las mejores oportunidades de venta. Incluye exploración, visualización y storytelling de datos, aplicando buenas prácticas de Data Science y control de versiones con Git/GitHub.
